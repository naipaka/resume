## 意欲・興味

- これまでの経験をフルに活かしつつ、サービスを成長させていきたい
- Flutter をメインにしつつバックエンドや Web 開発も経験してきたので、今後も幅を広げていきたい
- 同じ志を持った仲間とワンチーム感を持って働きたい
- チームメンバーの成長支援と技術力向上に貢献したい

## 価値観

- 仕事をしていて一番幸せを感じるのは、ユーザーのためになっていると実感できたとき
  - ユーザーからいい評価をもらったときは、このために生まれてきたのだと思える瞬間
- プロダクトを使ってくれているユーザーからフィードバックをもらうのが好き
  - 自分が考えつかなかったことや新しい視点を得ることに喜びを感じる
- 自分で自信を持って出したプロダクトに関しては、競合に負けたくないというモチベーションで開発
  - 個人開発で出しているアプリのうち 2 つは、主要な検索キーワードで 1 位を獲得
- 業務外での開発を通じて得た知見を共有してチームに貢献することが好き
- 仕事の効率を上げるために、隙間時間で役に立ちそうなツールを作ったりするのが好き
- 心理的安全性が高い場所だと、仕事のパフォーマンスを発揮できる
  - 具体的には、誰もが意見を言える、レビューの指摘にトゲがない、チームのメンバーを信頼できるなどを重視
- チーム一団となってプロダクトをよりよくしていくことに憧れている
- ワンチームで目標やビジョンに向かって進んでいきたい
  - そのため、現在の会社では明確な MVV の策定を提言したり、目標やロードマップの策定を提案
