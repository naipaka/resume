## 職務経歴

モバイルアプリ開発を中心に7年以上のエンジニア経験があります。

大手 SIer でのシステム開発からキャリアをスタートし、2年目で個人開発がきっかけでモバイルアプリ開発に転向しました。

その後、技術選定、アーキテクチャ設計、CI/CD 環境構築、コードレビュー基準策定などを担当し、開発チームが効率的に開発できる環境づくりを意識してきました。

副業や個人開発も並行して行い、MAU1 万人を超えるアプリの運用や OSS への貢献、技術記事の執筆、勉強会登壇などを通じて得られた知見を外部に共有しています。

### Altive 株式会社（2023/04〜現在）

Flutter/Dart を中心に、自社開発・受託開発の双方で 0→1 の新規開発から大規模リプレイスまで担当してきました。

CI/CD、Custom Lint、開発支援ツール作成など、技術基盤の整備を継続的に担い、チーム全体の開発効率と品質向上に取り組んでいます。

新人教育やレビュー基準の整備、MVV 策定提案、ロードマップ作成支援など、組織づくりにも関わり、プロジェクトとチームの両面で環境改善を進めてきました。

#### 技術面での主導的な役割

- CI/CD の構築・運用（GitHub Actions / Xcode Cloud）
- Custom Lint の作成によるコード品質の標準化
- OpenAPI を用いた API Client / Postman Collection の自動生成
- Dart CLI を用いた開発支援ツールの作成

#### 開発プロセス改善

- 質問テンプレート整備と短期フィードバックによる新人の立ち上げ支援
- モブプログラミングレビュー制度の導入
- Issue 消化速度・PR マージ速度の改善に向けた運用整備

#### 組織改善への取り組み

- Mission / Vision / Value（MVV）の策定提案
- 経営ロードマップの作成支援
- Gather・偏愛マップ導入によるコミュニケーション改善
- 新規メンバーのオンボーディング支援


### 株式会社ライトコード（2020/01〜2023/04）

大規模アプリ開発やプロトタイプ開発、新規サービス立ち上げなど多様なプロジェクトに参画しました。

CI/CD 環境構築、Flutter 未経験者への研修プログラム策定、コードレビュー基準整備を担当しました。

レビュアの負担偏りやメンバーのスキル向上が進まない課題に対してランダムレビュア制度の導入なども行いました。

### NEC ソリューションイノベータ株式会社（2018/04〜2019/12）

大手 SIer でのエンタープライズシステム開発に従事。Java や TypeScript を用いた BtoB システム開発を経験し、要件定義から設計、実装、テストまでの一連のプロセスを習得しました。

並行して個人で iOS アプリ開発のスキルを磨き、後のモバイルアプリ開発キャリアの基礎を築きました。

## 意欲・興味

- これまでの経験を活かし、サービスを成長させたい
- Flutter を中心に、バックエンドや Web 開発など、プロジェクトに応じて幅を広げたい

## 価値観

- ユーザーのためになっていると実感できたとき、仕事をしていて一番幸せを感じる
- プロダクトを使ってくれているユーザーからのフィードバックを大切にしている
- 自信を持って出したプロダクトに関しては、競合に負けたくないというモチベーションで開発している（個人開発アプリのうち2つは主要な検索キーワードで1位を獲得）
- 業務外での開発を通じて得た知見をチームに共有し、貢献することを重視
- 仕事の効率を上げるために、隙間時間で役に立ちそうなツールを作ることも楽しむ
- 心理的安全性が高い場所で、仕事のパフォーマンスを発揮できる（誰もが意見を言える、レビューの指摘にトゲがない、チームのメンバーを信頼できるなどを重視）
- チーム一丸となってプロダクトをよりよくしていくことに憧れがある
- ワンチームで目標やビジョンに向かって進んでいきたい（現在の会社では MVV の策定を提言したり、目標やロードマップの策定を提案）
